Question 3

Length of Last Word

Given a string s consisting of uppercase, lowercase letters and spaces, return the length of the last word in the string.

Note: A word is a maximal substring consisting only of non-space characters.

Example 1:

Input: s = "Hello World"
Output: 5
Explanation: The last word is "World" with length 5.

Example 2:

Input: s = " fly me to the moon "
Output: 4
Explanation: The last word is "moon" with length 4.

Constraints:

  • 1 <= s.length <= 104
  • s consists only of English letters and spaces ' '.
